You can apply differrent horizontal spacing and vertical spacing. An example is border-spacing: 15px; 25px; for 15px horizontal spacing and 25px for vertical spacing. Horizontal spacing refers to the width of the gap between adjacent cells in the same row. Vertical spacing refers to the height of the gap between adjacent cells in the same column. The padding property adds space inside each cell's border.

The margin has no effect to specify space outside an element's border. Use the border-spacing property to specify consistent horizontal and vertical gaps between table cells

If you want the table layout where content controls the table's cells, use the CSS display property. Use CSS position properties for fixed table cell sizes.

Browsers use a default layout scheme known as normal flow. Normal flow is block elements are laid out top to bottom, one block element per row, and phrasing elements are laid out from left to right. Use CSS position properties to remove an element from the normal flow. Use the position property with the value absolute to change the layout scheme from the default normal flow to absolute positioning. Use top and left to specify the target's top left corner is position relative to the element's containing block.

The width and height properties. If you don't specify the element's width, then the element's width is determined by the natural width of the element's content. The height property is like the width property you may want to omit such as images. If you want absolute positioned boxes to be aligned vertically at their bottoms, then for each box include the top and height.

Here are the CSS position properties.

  • position: absolute. Tells the browser to postion the element relative to the containing block. Normally, the containing block is the web page's html element.
  • left: value in px or percentage. Specifies the distance between the containing block's left edge and the element's left edge. The left edge is the left of any padding, border, or margin.
  • top: value in px or percentage. Specifies the distance between the containing block's top edge and the element's top edge. The top edge is the above any pading, border, or margin.
  • width: value in px or percentage. Specifies the width of the invisible box which constrains the element's contents. The width doesn't include padding, border, or margin.
  • height: value in px or percentage. Specifies the height of the invisible box which constrains the element's text. The height doesn't include padding, border, or margin.
