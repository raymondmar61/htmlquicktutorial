Pseudo-class selector. It's elements in a certain state belong to the same class. However, the class name isn’t in the markup—it’s something the browser just keeps track of. Pseudo-class selectors are indicated by the colon (:) character. They typically go immediately after an element name—for example, li:first-child.

Link colors is an example of pseudo-class. a:link is unvisited link. a:visited is visited link or link user clicked. a:focus is user clicks links. a:hover is user hovers mouse on link. a:active is browser is going to the link. For example, a:hover {color: maroon; background-color: #ffd9d9;} link is pink background color when the mouse hovers over link. When you apply styles to a elements with all five pseudo-classes, the required order for pseudo-classes is: :link :visited :focus :hover :active. The order in which they appear is important for them to function properly; for example, if you put :link or :visited last, they override the other states, preventing them from appearing.
