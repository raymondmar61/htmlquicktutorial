The figure element. A figure holds text, programming code, an illustration, a picture, or a data table. The red border shows the figure expansive equal size margins at the left and the right. The browser generates the left margin and right margin by default.

Organizational elements. Group web page content into sections to use CSS and JavaScript to manipulate their content. There's usually no need to use these organizational elements for small web pages. The organizational elements are section, article, aside, nav, header, and footer. Organizational elements are block elements. They span the entire width by default. Use the CSS float to float left or float right. Floated elements are positioned on the same line as the content which precedes it or on the next line if the preceding content is a block element.
