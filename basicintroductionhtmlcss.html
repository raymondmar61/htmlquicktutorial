Inline style or inline CSS style or CSS style at the html tag. An inline style should be used when a unique style is to be applied to a single occurrence of an element. To use inline styles, you use the style attribute in the relevant tag. The style attribute can contain any CSS property. style is style="color: green; margin-left: 30px".

Inline style tags can be used in link, div which defines a section, and span which also defines a section.

Core attributes class, id, style, and title. Not valid in base, head, html, meta, param, script, style, and title elements html elements. Class is a class rule or style rule for multiple html elements. Id is an id name for a unique html element; in other words, used once. Style is an inline style to define CSS at a html element. Title is a toolip or text to display as a tool tip.

Language attributes dir and lang. Not valid in base, br, frame, frameset, hr, iframe, param, and script elements. Dir sets the text direction. Lang sets the language code.

Keyboard attributes accesskey and tabindex. Accesskey sets a keyboard shortcut to access an element. Tabindex sets the tab order of an element.
