CSS rules apply to elements in a web page. There are differente types of selectors. The chapter introduces type selectors and the universal selector. The type selector uses an element type such as hr to match all instances of the hr element type. Type selector example hr {width: 50%} applies a 50% width to all hr elements. Universal selector example * {text-align: center} centers text for all text-oriented element. The asterisk * is a wildcard which matches every element in a web page.

If you want to apply the same formatting feature(s) to more than one type of element, separate each element with a comma for which the elements appear as part of a comma separated list. For instance,

        * {
            text-align: center;
        }
        h2, p {
            font-style: italic;
            color: blue;
        }

aligns every text centered. The <h2>, <p> italics and blue all text.

Coding convention suggest <style> positioned at the bottom of the <head> or above the </head>. CSS style format is selector(s) {property1: value1; property2; value2; property3: value3; property#: value#;} for which each property on its own line or block formatting.

The class selector is a dot or period. Class selector selects elements wich a particular value for their class attribute. The paragraph class attribute is redparagraph. Common CSS properties. A CSS property specifies one aspect of an element appearance. There are more than a hundred CSS properties. Remembering all the properties is daunting. Need a CSS reference. The Mozilla Developer Network more user-friendly CSS reference https://developer.mozilla.org/en-US/docs/Web/CSS/Reference. The chapter discusses five properties: color, font, text, border, and margin/padding.

Color properties: color, background-color. Use color for font color or text color. Examples: .classfontcolor {color: red;} .classrgbvalues {background-color: rgb(52%,20%,45%); The Red, Green, and Blue RGB(red number, green number, blue number) use percentage 0%-100%, integer 0 to 255, or hexadecimal 00 to ff. black is rgb(0%,0%,0%). white is rgb(100%,100%,100%). blue is background-color: #0f42ba; using hexadecimal. rgba is red, green, blue, and opacity. Opacity determines how opaque the color or the inability to see through. Two formats rgba(red-integer,green-integer,blue-integer,opacity-number-between-0-and-1) and rgba(red-percent,green-percent,blue-percent,opacity-number-between-0-and-1). For example, .yellow-bg2 {background-color: rgba(255,255,0,.4);} .yellow-bg3 {background-color: rgb(255,255,0); opacity: .4;}

Font properties: font-style, font-variant, font-weight, font-size, font-family.

  • font-style displays text normally or slanted. font-style values normal, oblique, and italic.
  • font-variant displays how lowercase letters are displayed. font-variant values normal and small-caps.
  • font-weight displays the text characters boldness. font-weight values normal, bold, bolder, lighter, and numbers 100 to 900 from thinnest 100 to thickest bold for which 400 is the same as normal and 700 is the same as bold.
  • font-size displays the size of the text characters. font-size values xx-small, x-small, small, medium, large, x-large, xx-large, smaller, larger, and number of em units for which one em is the height of the element's normal font size. For example, .hugefont {font-size: xx-large;} .emexampletriplenormalfontsize {font-size: 3em}.
  • font-family displays the font type. font-familiy values monospace, serif, sans-serif, cursive, fantasy. For example, .usethesefonts {font-familiy: "New Century Schoolbook", Times, serif;}
  • The font shorthand property is font: [font-style value] [font-variant value] [font-weight value] font-size value[/line-height value] font-family value;. font-size value and font-family value are required. For example, .largeitalic {font: italic large "Arial Black", Helvetica, sans-serif;} for which commas separate the font-family values. The commas are delimiters for a sublist. The font-family property is a sublist.
    • line-height displays vertical separation between each line of text in an element. line-height: 2em; display its lines with a vertical separation equal to twice the height of a normal character. An em is the height of a normal character. For example, .doublespace {line-height: 2em;} .fontsizelineheighttogehter {font: 1em/2em;} for which the font 1em is normal size and the line spacing 2em is double spaced.
    • Use a percentage value. .proof-reading {line-height: 300%;} The percentage value display line height three times the height of a normal character the same as 3em.

Text properties: text-align, text-decoration, text-transform, text-indent.

  • text-align specifies the horizontal alignment. text-align values left, right, center, and justify.
  • text-decoration specifies decorative. text-decoration values none, underline, overline, line-through RM: which is strike through, and blink.
  • text-transform specifies the capitalization. text-transform values are none, capitalize, uppercase, and lowercase. capitalize transform the first character of each word to uppercase or titlecase.
  • text-indent specifies the size of indentation of the first line in a block of text. If you want to adjust all the lines in a block of text, use the margin property. The best way to specify a value is em units. For example, p {text-indent: 4em;}

Border properties: border-style, border-width, border-color, shorthand.

  • border-style specifies the border type. border-style values none, solid, dashed, dotted, and double.
  • border-width specifies the width. border-width values thin, medium, thick; or number of px units. A CSS pixel unit is the size of a single projected dot on a computer monitor at 100% zoom on the monitor. border-width must be used with border-style. Different widths for different border sides can be done. For example, .differentsolidborders {border-style: solid; border-width 4px 2px 0px 2px;} The border's four sides in clockwise order starting from the top. Top sie 4px value, right 2px, bottom no border, left 2 px. .leftright2pxsizeborders {border-style: solid; border-width 4px 2px 0px;} .topbottom4pxleftright2pxsizeborders {border-style: solid; border-width 4px 2px;}
  • border-color specifies the color border. border-color must be used with border-style.
  • border as shorthand or compact. Specify the border's width, style, and color in order. .thindottedblueborder{border: thin dotted blue;} .solid10pxborder{border: 10px solid;}
  • border-radius specifices the curvature at the four corners.

Element box. Padding property and margin property. Create gaps around elements. Every web page element has an element box. An element box has a border, padding inside the border, and a margin outside the border. Most elements the default border, padding, and margin widths are zero. Adjust the widths with the border-width, padding, and margin properties. Think of a delivered mail package. You put the padding on the inside of the border package; therefore, the margin is the outside of the border package. Usually use px values for padding and margin. .solidborder20pxinsideoutside {border: solid; padding: 20px; margin: 20px;} Can specifiy different padding widths for the four sides. Can use separate padding side properties padding-top, padding-right, padding-bottom, and padding-left. Likewise for margin side properties margin-top, margin-right, margin-bottom, and margin-left. Negative numbers for margin and padding cause an overlap.

display: inline-block; is a must for vertical margins. The CSS creates a blockish.

Different length units. Length numbers.

  • Use em for font-related properties such as font-size.
  • Use px for fixed properties such as border properties and layout.
  • Use percentages % for growing and shrinking properties such as margin properties.
  • Use absolute sparingly for fixed sizes.
