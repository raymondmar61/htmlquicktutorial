The chapter teaches CSS concepts and CSS basic syntax. Apply the CSS rules to elements. Type the CSS rules at the top of the web page's or in an external file. Describe CSS properties to change appearances of the elements. Introduce CSS properties for color, font, line height, borders, padding, and margins.

CSS rules apply to elements in a web page. There are differente types of selectors. The chapter introduces type selectors and the universal selector. The type selector uses an element type such as hr to match all instances of the hr element type. Type selector example hr {width: 50%} applies a 50% width to all hr elements. Universal selector example * {text-align: center} centers text for all text-oriented element. The asterisk * is a wildcard which matches every element in a web page.

If you want to apply the same formatting feature(s) to more than one type of element, separate each element with a comma for which the elements appear as part of a comma separated list. For instance,

        * {
            text-align: center;
        }
        h2, p {
            font-style: italic;
            color: blue;
        }

aligns every text centered. The <h2>, <p> italics and blue all text.

Coding convention suggest <style> positioned at the bottom of the <head> or above the </head>. CSS style format is selector(s) {property1: value1; property2; value2; property3: value3; property#: value#;} for which each property on its own line or block formatting.

The class selector is a dot or period. Class selector selects elements wich a particular value for their class attribute. The paragraph class attribute is redparagraph. You can have many comma separated selectors sharing the same property-value pairs. Also, no spaces naming the class selector. Use hyphens.

The class selector with element type prefixes is a dot or period with the html element to the left of the dot or period. CSS style format is elementtype.classvalue {property1: value1; property2; value2; property3: value3; property#: value#;} for which each property on its own line or block formatting. A class selector can use the universal selector * instead of indicating the element. The universal selector matches all elements. *.classvalue {property1: value1; property2; value2; property3: value3; property#: value#;}. For example, *.bigwarning {font-size: x-large; color: red;} matches all elements with the class attribute value bigwarning.
