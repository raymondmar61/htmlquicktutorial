Two drawbacks. The inherit flexibility can behave nonsensical. It's not suitable for large-scale layouts; best for components of a page such as header or sidebar, or small-scale layouts.

Containers and items are the two kinds of page objects working with flexbox. A flex container is any type of parent block element which surrounds one or more elements such as div, p, and body. A flex item is any type of child elements.

The CSS declaration is display: flex; The container's child elements automatically become flex items. Flexbox containers have two axes which are main and cross. Main is the axis which runs in the same direction as the container's items. Cross or secondary axis is the axis which runs perpendicular to the main axis.
