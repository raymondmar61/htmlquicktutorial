The figure element identifies content that illustrates or supports some point in the text. A figure may contain an image, a video, a code snippet, text, or a table. Although you can simply add an image to a page, wrapping it in figure tags makes its purpose explicitly clear semantically. It also works as a hook for applying special styles to figures but not to other images on the page.

div using an id attribute instead of class attribute. The id attribute is an unique identifier used once in a page; for example, ISBN numbers to write HTML code for books. The class attribute may be shared by multiple elements; for example write HTML code for books formatting: <div id="ISBN0321127307" class="listing"> and <div id="ISBN0881792063" class="listing">. id and class must contains one character and no spaces.
