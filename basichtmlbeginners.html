Definitions

  • Cascading: page elements styled depending on rules which control which style takes precedence when there are more than one style for an element.
  • Cascading Style Sheet CSS: Statements or rules specify how content of an element appears. Cascading is rules on precedence. 1 Later properties override earlier properties. 2 More specific selectors override less specific selectors. 3 Specified properties override inherited properties. The rule cascades down from general to specific. For example, a p tag within the div tag with font type is chosen over a div tag with font type.
  • Inheritance: certain styles are inherited down the document tree; for example, a font-family in the body style means all text on the page have the font-family.
  • In-line styles: a method to place styles directly on the element instead of the head of the document or a style sheet.
  • Class: a class to style more than one element.
  • ID: an id to style one element.
  • Elements: a component of a web page; for example, <h1> <h2> <h3>.
  • Tags: define the start and the end of an element which are the less than tag < and greater than tag >; for example, <h1> and <p>.
  • Attributes: properties of an element added in the element; for example, the url for the href and the image name for the img.
  • POST and GET: send data to another webpage to process the data used in forms.
  • HTML: hypertext Mark Up Language the text to represent information on a web browser.
  • A style is rules defining a particular attribute. Styles can be written in line on an HTML element tag, in the header, or a separate style sheet file.
  • Block elements: Block level elements start a new line. <h1>, <h2>, <p>, <div> are examples.
  • Inline elements: In-line elements continue on the same line. <i>, <strong>, <span> are examples.

Avoid using points and picas. Pixels are fixed size units in screen media read on a screen. One pixel is one dot on the screen. Ems is a scalable unit in web document media. An em is equal to the current font size; for example, if the font size is 12pt, 1em is equal to 12pt, 2em is 24pt, .5em is 6pt. 1em is the height of the letter "M". Percent is like the em. CSS Box Model

Every element is considered a rectangular box. The properties for a CSS box model is the content, padding, border, and margin. The padding is around the content. The padding is a gutter around the content. The border is a line around the padding or a line on the outside of the padding. The margin creates spacing between boxes. The padding adds space between the content and the border. The margin adds space between the border and other content.

The width and height assigned to the box refers to the content only. The padding, borders, and margins are added to give the total box width. For example, a 100px wide box is defined {margin: 10px; padding: 10px; width: 60px} left margin 10px, left padding 10px, width of the content 60px, right padding 10px, and right margin 10px 10+10+60+10+10=100.
