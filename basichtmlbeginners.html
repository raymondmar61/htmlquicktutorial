A basic web page consists of a declaration, head, and body. The !DOCTYPE declaration is not an HTML tag. The !DOCTYPE is an instruction to the web browser what version of HTML used. <!DOCTYPE html> is HTML5. <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd"> is HTML 4.01.

Definitions

  • Cascading: page elements styled depending on rules which control which style takes precedence when there are more than one style for an element.
  • Inheritance: certain styles are inherited down the document tree; for example, a font-family in the body style means all text on the page have the font-family.
  • In-line styles: a method to place styles directly on the element instead of the head of the document or a style sheet.
  • Class: a class to style more than one element.
  • ID: an id to style one element.
  • Elements: a component of a web page; for example, <h1> <h2> <h3>.
  • Tags: define the start and the end of an element which are the less than tag < and greater than tag >; for example, <h1> and <p>.
  • Attributes: properties of an element added in the element; for example, the url for the href and the image name for the img.
  • POST and GET: send data to another webpage to process the data used in forms.
  • HTML: hypertext Mark Up Language the text to represent information on a web browser.
