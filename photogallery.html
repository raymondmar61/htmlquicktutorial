The body CSS code is a single-column container which gives a vertical main axis--flexbox container. Items aligned with the start of the vertical main axis or the top of the page. Centered horizontally. Footer appears at the bottom of the screen.

Set minimum height min-height: 100vh;. vh is the viewport height unit where 100vh equals 100% of the current viewport height. The footer section always appear at the bottom of the screen.

The header is a flexbox container. The header uses the full width of the window.

The navigation is an unordered list of links flexbox container centered horizontally and vertically. The unordered list doesn't have bullets. The CSS color: inherit on the a element tells the browser to use the parent's color for the links.

Further, add a span element for the navigation link for which the user is currently on its page. Also, add CSS code when the user hovers over the link.

Main is a flexbox container centers the thumbnails horizontally and wrap.

The footer is aligned horizontally and vertically using flex container. Set the main element flex-grow to 1 to force main to fill space and allow footer to appear at the bottom even wthen the main element doesn't fill the browser window vertically. Set align-content: flex-start to elimiate weird vertical spacing in the thumbnails.

Fixed footer. Add padding-bottom: 3.5em; in CSS main. Add position: fixed; bottom: 0; in CSS footer.
